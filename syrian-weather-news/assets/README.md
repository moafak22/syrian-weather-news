# Assets Directory

This directory contains all media files for the Syrian Weather News website.

## Directory Structure

```
assets/
├── images/          # Image files
├── videos/          # Video files
├── audio/           # Audio files
├── scripts/         # Script documents (PDF)
└── README.md        # This file
```

## Required Files

### Images (`assets/images/`)
Place the following image files in this directory:

- `video-poster.jpg` - Poster/thumbnail for the weather video
- `presenter.jpg` - Photo of the weather presenter
- `editor.jpg` - Photo of the video editor
- `designer.jpg` - Photo of the graphic designer
- `production-shot-1.jpg` - Behind-the-scenes production photo
- `production-shot-2.jpg` - Recording session photo
- `weather-map.jpg` - Syria weather map graphic
- `graphics-sample.jpg` - Sample weather graphics
- `team-work.jpg` - Team collaboration photo
- `editing-suite.jpg` - Post-production workspace photo

**Recommended specifications:**
- Format: JPG or PNG
- Resolution: 1920x1080px or higher
- File size: < 2MB per image for optimal performance

### Videos (`assets/videos/`)
Place the following video files in this directory:

- `final-weather-report.mp4` - Main weather report video

**Recommended specifications:**
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080px (Full HD)
- Frame rate: 30fps
- File size: Compressed for web delivery

### Audio (`assets/audio/`)
Place the following audio files in this directory:

- `opening-theme.mp3` - Opening theme music
- `background-music.mp3` - Background ambient music
- `voiceover.mp3` - Professional voiceover recording

**Recommended specifications:**
- Format: MP3
- Bitrate: 128-192 kbps
- Sample rate: 44.1 kHz

### Scripts (`assets/scripts/`)
Place the following script documents in this directory:

- `main-weather-script.pdf` - Complete daily weather forecast script
- `opening-script.pdf` - Program opening remarks script
- `regional-script.pdf` - Regional forecasts script

**Recommended specifications:**
- Format: PDF
- File size: < 5MB per document

## Placeholder Content

Currently, the website references these files but they are not included in the repository. You can:

1. **Add your own media files** matching the names above
2. **Use placeholder images** - The website will display gray backgrounds if images are missing
3. **Replace with your content** - Update the HTML if you have different file names

## Image Placeholders

If you need placeholder images for testing, you can use:
- **Placeholder services**: https://placeholder.com or https://picsum.photos
- **Create colored rectangles** in any image editor

Example placeholder sizes:
- Team photos: 400x400px (square)
- Production shots: 1200x800px (landscape)
- Video poster: 1920x1080px (landscape)

## Notes

- All media files should be optimized for web delivery
- Consider using WebP format for images (with JPG fallback)
- Compress videos before uploading
- Keep total asset size reasonable for fast page loading
- Add descriptive alt text in HTML for accessibility
